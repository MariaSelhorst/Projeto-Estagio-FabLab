# Sistema de Inventário FabLab

## Sobre o Projeto

O Sistema de Inventário FabLab é uma aplicação web desenvolvida para facilitar o controle e a gestão dos recursos disponíveis no laboratório. O sistema permite o cadastro de itens, o acompanhamento de movimentações de entrada e saída de estoque, além do gerenciamento de empréstimos realizados pelos usuários.

O objetivo principal é fornecer uma solução simples, organizada e eficiente para o controle do inventário, garantindo maior precisão no acompanhamento dos materiais e equipamentos disponíveis.

## Funcionalidades

- Cadastro, edição, visualização e remoção de itens;
- Controle de estoque em tempo real;
- Registro de entradas e saídas de materiais;
- Controle de empréstimos e devoluções;
- Histórico de movimentações;
- Consulta do estoque atual;
- Identificação dos responsáveis pelos empréstimos.

## Regras de Negócio

- Não é permitido registrar saídas superiores à quantidade disponível em estoque;
- Entradas aumentam automaticamente a quantidade em estoque;
- Saídas reduzem automaticamente a quantidade em estoque;
- Itens emprestados permanecem registrados até sua devolução.
